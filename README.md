# Praktikum 3

Manajemen Konten (Halaman Publik): Implementasi daftar artikel dinamis yang menampilkan judul, gambar placeholder, dan cuplikan teks.

Fitur "Baca Selengkapnya" (slug URL) yang mengarahkan ke detail artikel spesifik (misal: artikel kedua).

Portal Admin: Pembuatan dashboard khusus admin pada path /admin/artikel untuk mengelola seluruh data artikel.

Terdapat tabel daftar artikel lengkap dengan kolom status dan aksi (Ubah/Hapus).

Fitur CRUD (Backend):

Tambah Data: Implementasi form /admin/artikel/add untuk input judul dan isi artikel baru ke database.

Update Data: Fitur edit artikel (/admin/artikel/edit/id) untuk memperbarui konten yang sudah ada tanpa membuat data baru.

Delete Data: Tombol hapus untuk membersihkan record artikel dari sistem.

Struktur UI: Konsistensi penggunaan layout header, sidebar widget, dan footer di seluruh halaman.

Pembedaan navigasi antara halaman publik (Home, Artikel, dsb) dengan navigasi panel admin (Dashboard, Tambah Artikel).
